# Welcome to the Claude Code Deep Dive

---

This intensive hands-on training will take you from Claude Code basics to advanced automation.

## What You'll Learn

- **Claude Code CLI**: Installation, commands, permissions, interactive mode
- **Configuration**: CLAUDE.md, project memory, settings
- **Full-Stack Development**: Build complete apps from the terminal
- **Database Operations**: Schema design, migrations, RLS policies
- **Git Integration**: Branching, PRs, code review workflows
- **Testing & Debugging**: Unit tests, integration tests, health checks
- **API Development**: REST APIs, webhooks, Edge Functions
- **UI/UX**: Design systems, components, responsive layouts
- **Automation**: Scripts, CSV processing, report generation
- **Advanced**: Slash commands, templates, performance optimisation

## Prerequisites

- A computer with terminal access
- GitHub account
- Claude account (Anthropic)
- Node.js installed
- Basic familiarity with the command line
